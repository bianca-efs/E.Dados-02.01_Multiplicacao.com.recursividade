Recursividade em Java – Introdução às Estruturas de Dados

Enunciado:
Criar uma aplicação em Java com uma **função recursiva** que receba dois valores inteiros, **A** e **B**, e retorne o resultado da **multiplicação de A por B usando apenas somas**. A função deve realizar a operação de forma recursiva, acumulando o resultado até atingir a condição de parada.


Objetivo

O objetivo deste exercício é praticar:
* Recursividade em Java
* Operações com números inteiros
* Identificação de condição de parada em funções recursivas
* Relação entre o termo atual e o resultado anterior na construção da solução recursiva
* Raciocínio lógico aplicado a operações matemáticas básicas


Requisitos

O programa deve:
1. Receber dois números inteiros (**A** e **B**) como parâmetros.
2. Calcular a multiplicação de A por B usando apenas somas, de forma recursiva.
3. Utilizar recursão sem loops (`for`, `while`, etc.).
4. Retornar o resultado como um número inteiro (`int`).
5. Incluir validação para garantir que os valores de entrada sejam inteiros.


Tecnologias:
* Oracle Java
* Programação Orientada a Objetos (POO)
* Recursividade

Observações
Este exercício faz parte dos estudos de **Introdução às Estruturas de Dados**, com foco no desenvolvimento do raciocínio lógico e na compreensão de funções recursivas aplicadas a operações matemáticas básicas.
